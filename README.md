## Client & Purpose

Make It Charm is a pop-up charm-making and bedazzling experience for birthdays, workshops, and small events.

The purpose of this website is to present the brand, explain how the experience works, showcase available packages, and allow users to inquire or book through a contact form.

## Pages Included

1. Landing Page (Home)

   - Popular packages preview

2. Supporting Page (Packages)

   - FAQ section

3. Form Page (Contact)
   - Booking/inquiry form
   - Includes labeled inputs and appropriate input types
